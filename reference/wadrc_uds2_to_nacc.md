# Translate WADRC UDS-2 variable names to NACC variable names

Translate WADRC UDS-2 variable names to NACC variable names

## Usage

``` r
wadrc_uds2_to_nacc
```

## Format

A named vector where names of entries correspond to WADRC variable names
as they appear in the UDS-2 REDCap database, and entries contain NACC
variable names.
