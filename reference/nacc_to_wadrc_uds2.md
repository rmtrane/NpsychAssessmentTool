# Translate NACC variable names to WADRC variable names (UDS-2)

Translate NACC variable names to WADRC variable names (UDS-2)

## Usage

``` r
nacc_to_wadrc_uds2
```

## Format

A named vector where names of entries correspond to NACC variable names,
and entries contain WADRC variable names as is in UDS-2 database.
