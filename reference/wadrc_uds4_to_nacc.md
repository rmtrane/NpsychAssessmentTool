# Translate WADRC UDS-4 variable names to NACC variable names

Translate WADRC UDS-4 variable names to NACC variable names

## Usage

``` r
wadrc_uds4_to_nacc
```

## Format

A named vector where names of entries correspond to WADRC variable names
as they appear in the UDS-4 REDCap database, and entries contain NACC
variable names.
