# Translate WADRC UDS-3 variable names to NACC variable names

Translate WADRC UDS-3 variable names to NACC variable names

## Usage

``` r
wadrc_uds3_to_nacc
```

## Format

A named vector where names of entries correspond to WADRC variable names
as they appear in the UDS-3 REDCap database, and entries contain NACC
variable names.
