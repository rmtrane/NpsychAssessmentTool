# Translate NACC variable names to WADRC variable names (UDS-4)

Translate NACC variable names to WADRC variable names (UDS-4)

## Usage

``` r
nacc_to_wadrc_uds4
```

## Format

A named vector where names of entries correspond to NACC variable names,
and entries contain WADRC variable names as is in UDS-4 database.
