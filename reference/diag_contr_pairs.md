# Etiology Specific Variables

Etiology Specific Variables

## Usage

``` r
diag_contr_pairs
```

## Format

Tibble with four columns: `presump_etio_diag` contains variable names
that indicate if disease is present; `contribution` contains variable
names that indicate if disease contributes to diagnosis; `other` gives
variable names that give description of other diagnoses; `disease` gives
human readable names for diseases.
