# Translate NACC variable names to WADRC variable names (UDS-3)

Translate NACC variable names to WADRC variable names (UDS-3)

## Usage

``` r
nacc_to_wadrc_uds3
```

## Format

A named vector where names of entries correspond to NACC variable names,
and entries contain WADRC variable names as is in UDS-3 database.
